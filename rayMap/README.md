# 地图组件

## 描述

- 本示例描述了地图组件的相关使用，具体属性请查看官方文档。

## 注意事项

- 在使用地图组件的时候请务必先阅读[原生组件说明](https://developer.tuya.com/cn/miniapp/component/native-component/native-component)，了解异层渲染的原生组件使用限制。
- 在map组件上弹窗等覆盖操作时，需要调用`ty.nativeDisabled`禁止原生组件区域原生手势管理。
